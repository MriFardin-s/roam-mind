
# 🌍 RoamMind — Discover & Manage Destinations

<img width="100%" alt="RoamMind Preview" src="https://github.com/user-attachments/assets/5e7284b2-59d7-41a1-8ac9-7a48d69726e5" />

**RoamMind** is a modern, full-stack travel and destination exploration web application built with **Next.js**, **React**, and **Tailwind CSS**. It allows users to explore breathtaking destinations around the world, filter through categories, and manage destination records seamlessly with an intuitive user interface.

---

## ✨ Features

* **🧭 Explore Destinations:** Browse through a curated list of travel spots with images, locations, and categories.
* **📱 Fully Responsive Design:** Tailored experience across Mobile, Tablet, and Desktop screens (includes dynamic Card views for mobile and detailed Tables for desktop).
* **🔐 Authentication & Role Management:** Secure sign-in and protected routes to ensure authorized access for administrative tasks.
* **🛠️ Manage Destinations:** Easily add, view, or delete destinations with real-time feedback via interactive toast notifications and confirm modals.
* **⚡ Modern UI/UX:** Clean aesthetics powered by Tailwind CSS with custom alert dialogs and smooth interactions.

---

## 🚀 Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Frontend Library:** [React](https://react.dev/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Notifications:** [React Hot Toast](https://react-hot-toast.com/)
* **Language:** JavaScript (ES6+)

## 🛠️ Getting Started

Follow these steps to set up and run the project locally on your machine.

### Prerequisites

Make sure you have Node.js installed on your system:
* **Node.js** (v18.0.0 or higher)
* **npm** / **yarn** / **pnpm**

### Installation

1. **Clone the Repository:**
   ```bash
   git clone [https://github.com/MriFardin-s/roam-mind.git](https://github.com/MriFardin-s/roam-mind.git)
   cd roam-mind

```

2. **Install Dependencies:**
```bash
npm install

```


3. **Set Up Environment Variables:**
Create a `.env.local` file in the root directory and add your variables:
```env
NEXT_PUBLIC_API_URL=your_api_url

```


4. **Run the Development Server:**
```bash
npm run dev

```


5. **Open Application:**
Open your browser and navigate to `http://localhost:3000` to view the app.
